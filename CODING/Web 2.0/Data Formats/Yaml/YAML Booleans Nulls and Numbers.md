# Booleans
1. `true`
2. `false`
3. `yes`
4. `no`
5. `on`
6. `off`
``` YAML
is_enabled: true
is_disabled: false
feature_flag: yes
use_cache: no
auto_restart: on
debug_mode: off
```

# Null Values
1. `null`
2. `~`
``` YAML
no_value: null
empty_value: ~
missing_value:
```

# Numbers
1. Integers
2. Floats
3. Scientific Notations
``` YAML
integer: 42
float: 3.1415
scientific: 1.2e6  # 1,200,000
negative: -10
```
##### Special Number Values
``` YAML
infinity: .inf
negative_infinity: -.inf
not_a_number: .nan
```

# Code
``` YAML
is_active: true
is_admin: no
timeout: 30
pi_value: 3.1415
big_number: 1.2e6  # Equivalent to 1,200,000
empty_value: null
```
