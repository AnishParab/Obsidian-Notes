# Block Scalar
1.  `|`
2. **Block Scalar** -> Preserves Newlines

``` YAML
description: |
  This is a multi-line
  YAML string that preserves
  line breaks.
```

# Folder Scalar
1. `>`
2. **Folder Scalar** -> Joins lines into a single line.

``` YAML
message: >
  This will be written as a single line
  without line breaks.
```
