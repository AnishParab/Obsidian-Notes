# Variables
- Follow **camelCase**
### Declaring, Assigning and Re-asingning Variables
``` php
<?php
	$name = 'Anish Parab';
	echo $name;

	$something = $name
?>
```

---
# Basic Data-Types
## Scalar Types
Variable contains one value to it.
- **string** ---> "My Name"
- **int** ---> 69.
- **float** ---> 69.69
- **bool** ---> false (0), true (1)

## Arrays
``` php
<?php
$array = array("Aryan","Bala","Aman","Akanksha");

$names = ["Anish","Onkar","Nandish"]
?>
```

## Objects
``` php
<?php
$object = new Car()
?>
```
