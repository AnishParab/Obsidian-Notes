# Basic Syntax
``` python
def function_name(parameters):
    # logic
    return result
```

---
# Parameters and Arguments
## Basic
``` python
def add(x, y):
    return x + y

add(2, 3)  # 5
```

## Default Arguments
``` python
def power(base, exp=2):
    return base ** exp

power(3)      # 9
power(3, 3)   # 27
```
