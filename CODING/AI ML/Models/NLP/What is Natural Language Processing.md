# What is NLP ?
- The whole idea is to make machines understand and respond using natural languages like english.

---
# Real World Applications
- Contextual Advertisements.
- Email Clients ---> Spam filtering, smart reply.
- Social Media ---> Removing Adult content, opinion mining.
- Search Engines.
- Chatbots.

---
# Common NLP Tasks
- Text/Document classification.
- Sentiment Analysis.
- Information Retrieval.
- Parts of Speech Tagging ---> noun, verb etc.
- Language Detection and Machine Translation.
- Conversational Agents.
- Knowledge Graph and QA Systems.
