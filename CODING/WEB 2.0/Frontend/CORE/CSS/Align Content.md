# What?
Aligns **multiple lines** of items along the **cross axis** (only effective if `flex-wrap` is used).

---
# Values
| Value           | Description                              |
| --------------- | ---------------------------------------- |
| `flex-start`    | Lines packed to the start.               |
| `flex-end`      | Lines packed to the end.                 |
| `center`        | Lines centered.                          |
| `space-between` | Evenly distributed lines.                |
| `space-around`  | Even spacing around each line.           |
| `stretch`       | Lines stretched to fill space (default). |
