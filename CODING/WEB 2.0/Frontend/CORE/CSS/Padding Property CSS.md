# Syntax
``` css
padding: top right bottom left;

```

Example :

| Example                        | Meaning                                |
| ------------------------------ | -------------------------------------- |
| `padding: 20px;`               | 20px padding on all sides.             |
| `padding: 10px 20px;`          | 10px top/bottom, 20px left/right.      |
| `padding: 5px 10px 15px;`      | 5px top, 10px left/right, 15px bottom. |
| `padding: 5px 10px 15px 20px;` | Top, Right, Bottom, Left (clockwise).  |

---
# Side-Specific Properties

| Property         | Example                 |
| ---------------- | ----------------------- |
| `padding-top`    | `padding-top: 15px;`    |
| `padding-right`  | `padding-right: 10px;`  |
| `padding-bottom` | `padding-bottom: 20px;` |
| `padding-left`   | `padding-left: 8px;`    |
