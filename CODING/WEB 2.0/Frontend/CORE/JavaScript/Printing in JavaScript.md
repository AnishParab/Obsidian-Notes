# `console.log`
``` js
console.log("Hello World")
```
- Built-in next line.

---
# `process.stdout.write`
``` js
process.stdout.write("Pikachu")
```
- **No** built-in next line.

---
# `console.table`
``` js
console.table({ city: "Goa" })
```
- Gives an actual table as output.

---
# `console.warn`
``` js
console.warn("Gives Colors")
```
- Gives some colors to the text in the console.
