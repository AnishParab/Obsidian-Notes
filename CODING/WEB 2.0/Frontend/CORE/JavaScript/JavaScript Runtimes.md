# Runtime

| Runtime         | Engine         | Target Use Case             | Special Feature                |
| --------------- | -------------- | --------------------------- | ------------------------------ |
| **Node.js**     | V8             | Server-side JS              | Huge npm ecosystem             |
| **Deno**        | V8             | Modern secure runtime       | Built-in TS, secure by default |
| **Bun**         | JavaScriptCore | Fast full-stack dev         | Built-in bundler/test runner   |
| **Nashorn**     | JVM-based      | Java integration (legacy)   | Runs on JVM                    |
| **GraalJS**     | GraalVM        | Polyglot + enterprise       | Multi-language support         |
| **Rhino**       | Java-based     | Legacy Java embedding       | 100% Java                      |
| **QuickJS**     | QuickJS        | Embedded systems, scripting | Ultra small footprint          |
| **JerryScript** | JerryScript    | IoT devices                 | Runs on microcontrollers       |
