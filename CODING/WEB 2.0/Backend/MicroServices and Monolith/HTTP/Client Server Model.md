# Client-Server Model
- _HTTP_ is a **Client-Server Protocol.**
- Messages sent by client are called requests.
- Messages sent by server as an answer are called responses.
![[client_server.excalidraw|500]]
> Between the Client and Server there are numerous entities, collectively called **proxies**.

---
# Client
- Any tool or program that initiates the request.

---
# Server
- Responds by serving the document as requested by the client.
- These may be collection of machines sharing the load **(load balancing).**
