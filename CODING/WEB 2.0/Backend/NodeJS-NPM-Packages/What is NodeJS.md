# What is Node.js
- **Node.js** is a **JavaScript runtime environment** built on **Chromeâ€™s V8 engine**.  
- It allows you to **run JavaScript outside the browser**, primarily on the **server (backend)**.
- Node.js is **not** a programming language or framework.

> **Node.js is Single-threaded**.

---
# Features
| Feature             | Description                                                  |
| ------------------- | ------------------------------------------------------------ |
| Runtime Env         | Executes JS outside the browser (e.g., server, CLI)          |
| Built on V8         | Uses Chrome's fast JavaScript engine                         |
| **Single-threaded** | **Uses event-driven, non-blocking I/O for scalability**      |
| Backend Capable     | Enables JS for servers, APIs, file systems, networking, etc. |
| Comes with `npm`    | Node Package Manager for installing packages                 |
