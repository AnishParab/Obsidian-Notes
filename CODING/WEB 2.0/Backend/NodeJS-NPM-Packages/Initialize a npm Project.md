# Create `package.json`
``` bash
npm init
```

OR

``` bash
npm init -y
```

---
# What is `package.json` ?
- Lists metadata and installed packages
- Essential for reproducibility and collaboration

---
# Create `node_modules/` folder
``` bash
npm install
```

---
# What is `node_modules/` ?
- Folder where all dependencies are installed
- Auto-created when `npm install` is run
