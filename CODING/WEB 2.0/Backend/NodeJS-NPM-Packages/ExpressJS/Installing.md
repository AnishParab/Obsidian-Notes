# Make Directory
``` bash
mkdir myapp
$ cd myapp
```

---
# `package.json`
``` bash
npm init

```

---
# Entry Point File
- Make `index.js` or `app.js` as the entry point of your file.

---
# Install Express and Save it in Dependencies List
``` bash
npm install express --save

```

---
# To install Express temporarily and not add it to the dependencies list
``` bash
npm install express --no-save

```
