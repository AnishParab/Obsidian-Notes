# Search Highlighting (`hlsearch`)

- Enable:
```vim
:set hlsearch
```

- Disable:
```vim
:set nohlsearch
```

---
# Clear highlights (temporary)

```vim
:nohlsearch
```

**Note**
- `:nohlsearch` does **not** change the option.
- Highlighting returns automatically on next search (`/`, `?`, `n`, `N`).
