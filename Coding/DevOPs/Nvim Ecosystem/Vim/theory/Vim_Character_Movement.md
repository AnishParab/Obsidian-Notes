# Find (`f` / `F`)

- `fx` → move to next `x` **to the right** on the line
- `Fx` → move to next `x` **to the left** on the line

---
# Till (`t` / `T`)

- `tx` → move **until before** next `x` to the right
- `Tx` → move **until after** next `x` to the left

---
# Repeat Find/Till

- `;` → repeat last `f/F/t/T` in same direction
- `,` → repeat last `f/F/t/T` in opposite direction
