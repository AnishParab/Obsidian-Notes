# Key Notation (Common Special Keys)

- `<CR>` → Enter / Return
- `<Esc>` → Escape
- `<Tab>` → Tab
- `<Space>` → Spacebar
- `<BS>` → Backspace
- `<Del>` → Delete
- `<Up>` / `<Down>` / `<Left>` / `<Right>` → Arrow keys
- `<Home>` → Home key
- `<End>` → End key
- `<PageUp>` → Page Up
- `<PageDown>` → Page Down

---
# Modifier notation

- `<C-x>` → Ctrl + x
- `<S-x>` → Shift + x
- `<A-x>` or `<M-x>` → Alt/Meta + x
