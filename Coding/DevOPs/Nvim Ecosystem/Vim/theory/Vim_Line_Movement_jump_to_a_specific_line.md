# Jump to a Line Number

- `{n}G` → go to line `n`
    - `33G` → line 33

### File start / end
- `gg` → first line
- `G` → last line
- `1G` → first line (same as `gg`, but more typing)

---
# Jump by Percentage

- `{n}%` → jump to `n` percent through the file
    - `50%` → middle of file
    - `90%` → near end

---
# Jump within the Visible Window

- `H` → top of screen (High / Home)
- `M` → middle of screen
- `L` → bottom of screen (Low / Last)
