# Vim Config File Location

# `~/.config/vim/init.vim`

k- User configuration file for **Vim** (especially when using the XDG config layout).
- Vim loads this at startup to apply your settings:
    - options (`set number`)
    - keymaps (`nnoremap ...`)
    - plugin config (if using a plugin manager)

**Related**
- Traditional Vim config:
    - `~/.vimrc`
- XDG config help:

```vim
:help vimrc
```
