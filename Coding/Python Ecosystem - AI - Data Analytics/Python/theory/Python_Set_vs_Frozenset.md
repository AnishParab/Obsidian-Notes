# Set vs. Frozenset Comparison

| Feature                                | `set`   | `frozenset`        |
| -------------------------------------- | ------- | ------------------ |
| Mutable                                | ✅       | ❌                  |
| Hashable (usable as dict key)          | ❌       | ✅                  |
| Supports add/remove                    | ✅       | ❌                  |
| Supports union/intersection/difference | ✅       | ✅                  |
| Syntax                                 | `{...}` | `frozenset({...})` |
