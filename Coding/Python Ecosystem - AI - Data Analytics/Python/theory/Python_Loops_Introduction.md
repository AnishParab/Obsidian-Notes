# Summary
- Loops used are `for` and `while`
- Loop through sequences using `range()`, `enumerate()`, and `zip()`.
- Control loop behavior using `break`, `continue`, and `else` clauses.

---
# Identify when to use `for` and `while` loop
- `for` loop is used for iterating over a list, or when we know something has an auto-ending.
