# **Mutable data types (can be changed after creation):**

## `LSDB`

* `list`
* `dict`
* `set`
* `bytearray`
* `user-defined classes` (if implemented with mutable attributes)

# **Immutable data types (cannot be changed after creation):**

* `int`
* `float`
* `bool`
* `str`
* `tuple`
* `frozenset`
* `bytes`
* `complex`
