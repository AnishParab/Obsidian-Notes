# Logical Operators

| Operator | Description                          | Example               | Output  |
| -------- | ------------------------------------ | --------------------- | ------- |
| `and`    | True if **both** conditions are true | `(5 > 2) and (3 < 4)` | `True`  |
| `or`     | True if **either** condition is true | `(5 < 2) or (3 < 4)`  | `True`  |
| `not`    | Inverts the Boolean value            | `not (5 > 2)`         | `False` |
