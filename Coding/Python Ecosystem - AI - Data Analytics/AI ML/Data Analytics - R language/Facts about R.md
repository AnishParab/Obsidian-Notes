# Interesting Facts about R Programming Language

- R is an **open-source programming language** used for **statistical analysis**, **data visualization**, and **predictive modeling**.  
- It comes with a **command-line interface** and runs on **Windows, Linux, and macOS**.  
- Designed by **Ross Ihaka** and **Robert Gentleman** at the **University of Auckland, New Zealand**, R is now maintained by the **R Development Core Team**.

---
# Key Facts
1. **Origin and Naming**
   - R is an **implementation of the S programming language**, incorporating **lexical scoping semantics** inspired by **Scheme**.  
   - The name “R” comes from the **first names** of its authors and as a **play on the name S**.  

2. **Programming Paradigms**
   - Supports both **procedural** and **object-oriented** programming.  
   - Procedural: procedures, modules, records.  
   - Object-oriented: classes, objects, generic functions.  

3. **Interpreted Language**
   - R is **interpreted**, not compiled — no separate compiler is needed.  
   - Scripts run directly, making experimentation fast and simple.  

4. **Extensive Packages**
   - Over **100,000 packages** available via **CRAN** and **GitHub**.  
   - Packages cover a wide range of tasks from **regression** to **Bayesian analysis**.  

5. **Popularity**
   - R is one of the **fastest-growing** data science languages.  
   - It’s the **most-used language after SQL**, with **70% of data miners** using it.  
   - Ranked **#7 on PYPL** and **#1 on Google Search** for advanced analytics.  
   - Over **3 million users worldwide**.  

6. **Powerful Packages**
   - **rmarkdown:** Generate reproducible Word documents and PowerPoint presentations.  
   - **dbplyr:** Connect easily to almost any database.  
   - **bigquery:** Work directly with Google BigQuery and other high-performance stores.  
   - **flexdashboard:** Build interactive web apps in a few lines of code.  
   - **rsconnect:** Host apps locally or on the cloud.  
   - **nessy:** Create retro NES-style Shiny apps (even games).  
   - **plumber:** Convert R functions into web APIs.  

7. **Community and Ecosystem**
   - Strong, **supportive global community**.  
   - Rich resources, tutorials, and forums for assistance.  

8. **Academic and Industry Use**
   - Widely used in **finance**, **healthcare**, **pharmaceuticals**, **marketing**, and **academia**.  
   - Common in research fields like **biology**, **psychology**, and **economics**.  

9. **Cross-Platform**
   - Runs seamlessly on **Windows**, **macOS**, and **Linux**.  
   - Consistent performance across all platforms.
