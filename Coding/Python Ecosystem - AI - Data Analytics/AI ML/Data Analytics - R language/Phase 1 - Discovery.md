# Traditional Technologies Used
- **Spreadsheets** (Excel, LibreOffice Calc) for initial data exploration.  
- **Relational Databases** (MySQL, PostgreSQL, Oracle DB) to examine structured historical data.  
- **Manual reports and business documents** for understanding domain context.  

---
# Modern Technologies Used
- **Data discovery platforms**: Alteryx, RapidMiner, Talend.  
- **Business Intelligence tools**: Power BI, Tableau, QlikView.  
- **Big Data Platforms**: Apache Hadoop, Apache Spark (for analyzing large datasets).  
- **Cloud Data Services**: AWS Data Lake, Google BigQuery, Azure Synapse.  
- **Collaboration & Documentation tools**: Notion, Confluence, Jupyter Notebooks for hypothesis documentation and sharing insights.  

> Modern technologies allow teams to **access, explore, and understand large and diverse datasets** quickly, while traditional methods are often useful for small-scale or historical data analysis.
