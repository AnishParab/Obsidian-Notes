# What is Data?
- Data refers to **quantities, characters, or symbols** on which operations are performed by a computer.
- It is **stored and transmitted** in the form of **electrical signals**.

---
# Tabular Representation of Memory Sizes

| **Unit**         | **Size (in Bytes)** |
| ---------------- | ------------------- |
| 1 Byte           | 8 bits              |
| 1 Kilobyte (KB)  | 1,024 Bytes         |
| 1 Megabyte (MB)  | 1,024 KB            |
| 1 Gigabyte (GB)  | 1,024 MB            |
| 1 Terabyte (TB)  | 1,024 GB            |
| 1 Petabyte (PB)  | 1,024 TB            |
| 1 Exabyte (EB)   | 1,024 PB            |
| 1 Zettabyte (ZB) | 1,024 EB            |
| 1 Yottabyte (YB) | 1,024 ZB            |
