# Types of Digital Data
Digital data can be categorized based on its structure and format:

---
## 1. Structured Data
- **Definition:** Data that can be stored, accessed, and processed in a **fixed format**.  
- **Example:** Tables in relational databases, spreadsheets.

---
## 2. Unstructured Data
- **Definition:** Data with an **unknown or flexible form/structure**.  
- **Example:** Text files, images, videos, audio files, social media posts.

---
## 3. Semi-Structured Data
- **Definition:** Contains **both structured and unstructured components**.  
- **Example:** XML files, JSON files, NoSQL documents.

---
# Challenges with Digital Data
- **Large volume of data** (both structured and unstructured).  
- **Key challenge:** Building platforms that can **ingest and process unstructured data** as easily as structured data.  

---
# Applications & Key Data Sources
- Big data analytics is used across multiple domains:  
  - **Business Intelligence:** Customer behavior analysis, sales prediction.  
  - **Healthcare:** Patient records, medical imaging.  
  - **Social Media:** Sentiment analysis, trend detection.  
  - **IoT Devices:** Sensor data, telemetry.  
  - **Finance:** Fraud detection, risk modeling.

---
# Use Cases for Big Data Analytics
- **Real-time decision making** using streaming data.  
- **Predictive analytics** for sales, marketing, and maintenance.  
- **Customer segmentation** and personalization.  
- **Anomaly detection** in systems or networks.  
- **Operational efficiency** improvements via data-driven insights.
