**Definition:**  
Dimensionality reduction is the process of **reducing the number of input features** while retaining essential information.

---
## Key Concepts
-  **Dropping Irrelevant Features**  
   - Some features provide little or no useful information.  
   - Example: Hair color when selecting a car.  
   - Requires estimating which features can be safely removed.

-  **Combining Features**  
   - Some features can be merged to **retain or enhance information**.  
   - Example: Total income of all family members for a loan application.  
   - Requires estimating which features can be combined.

---
## Benefits
- Reduces **computation time** and **memory usage**.  
- Simplifies **model explanations**.  
- Mitigates the **curse of dimensionality**.
