# What is a Large Language Model (LLM) ?
- A large language model is an artificial intelligence system trained to understand and generate **human language**.
- It uses **machine learning**, particularly **deep learning with neural networks**.

---
# What it can do ?
- Answer questions
- Write code
- Summarize texts
- Translate languages
- Generate creative writing
- Assist in research or customer support

---
# Examples of large language models:
* **GPT-5** (OpenAI)
* **Claude 3** (Anthropic)
* **Gemini 1.5** (Google DeepMind)
* **LLaMA 3** (Meta)
* **Mistral Large** (Mistral AI)
* **Command-R+** (Cohere)
* **Yi 1.5** (01.AI)
* **ERNIE Bot 4.0** (Baidu)
* **Tongyi Qianwen 2.0** (Alibaba)
* **Jurassic-2** (AI21 Labs)

All are **transformer-based** and trained on large text corpora for general-purpose reasoning, generation, and comprehension.
