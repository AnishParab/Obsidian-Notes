# Hover/Focus/State Prefixes
| Prefix         | Description                   |
| -------------- | ----------------------------- |
| `hover:`       | On mouse hover                |
| `focus:`       | When input is focused         |
| `active:`      | On active state (clicking)    |
| `disabled:`    | When input/button is disabled |
| `group-hover:` | When parent group is hovered  |

---
# **Example**
``` html
<button class="hover:ml-2 hover:px-4">
This applies extra margin-left and padding-x only on hover.
</button>
```

---
# `hover:`
- **Description**: Applies styles **when the mouse pointer is over** the element.

---
# `focus:`
- **Description**: Triggers styles when an element **receives keyboard or mouse focus**.

---
## `active:`
- **Description**: Applies styles when the element is **being clicked** (or tapped).

---
# `disabled:`
- **Description**: Styles elements that have the `disabled` attribute.

---
# `group-hover:`
- **Description**: Allows styling of **child elements** when the **parent (with `group`) is hovered**.
