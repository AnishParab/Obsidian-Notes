# `text-*`
- Tailwind's `text-*` classes are used to style **text content** such as headings, paragraphs, labels, etc.

---
# Purpose
The `text-*` prefix lets you control:
- **Text alignment**
- **Font size**
- **Text color**
- **Font weight**
- **Decoration**
- **Transformations**

---
# Common `text-*` Utilities
### **Text Alignment**
| Class          | Description             |
| -------------- | ----------------------- |
| `text-left`    | Align text to the left  |
| `text-center`  | Center the text         |
| `text-right`   | Align text to the right |
| `text-justify` | Justify the text        |

---
### **Text Size**
| Class        | Font Size | Pixels |
| ------------ | --------- | ------ |
| `text-sm`    | 0.875rem  | 14px   |
| `text-base`  | 1rem      | 16px   |
| `text-lg`    | 1.125rem  | 18px   |
| `text-xl`    | 1.25rem   | 20px   |
| `text-2xl`   | 1.5rem    | 24px   |
| ...and so on |           |        |
Use `text-lg` for slightly larger text than default.

---
### **Text Color**
| Example         | Description      |
| --------------- | ---------------- |
| `text-blue-400` | Light blue color |
| `text-red-600`  | Dark red color   |
| `text-gray-700` | Medium gray      |
Tailwind supports **500+ colors and shades**.

---
### **Font Weight**
| Class         | Description       |
| ------------- | ----------------- |
| `font-light`  | Light weight text |
| `font-normal` | Normal weight     |
| `font-bold`   | Bold text         |
| `font-black`  | Extra bold        |

---
### **Text Decoration & Transform**
| Class          | Description             |
| -------------- | ----------------------- |
| `underline`    | Underlines the text     |
| `line-through` | Strikes through text    |
| `uppercase`    | Transforms to UPPERCASE |
| `lowercase`    | Transforms to lowercase |
| `capitalize`   | Capitalizes each word   |
