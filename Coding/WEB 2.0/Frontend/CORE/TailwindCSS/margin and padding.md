# Syntax Overview
- `m` → margin
- `p` → padding

---
# Direction Suffixes
| Direction | Suffix | Meaning                   |
| --------- | ------ | ------------------------- |
| (none)    | `m-4`  | All sides                 |
| `t`       | `mt-4` | Top only                  |
| `r`       | `mr-4` | Right only                |
| `b`       | `mb-4` | Bottom only               |
| `l`       | `ml-4` | Left only                 |
| `x`       | `mx-4` | Horizontal (left & right) |
| `y`       | `my-4` | Vertical (top & bottom)   |

---
# Margin Utilities
| Class     | Description                 |
| --------- | --------------------------- |
| `m-4`     | `margin: 1rem` on all sides |
| `mt-2`    | `margin-top: 0.5rem`        |
| `mb-4`    | `margin-bottom: 1rem`       |
| `mx-auto` | Auto horizontal centering   |
| `-ml-4`   | Negative left margin        |

---
# Padding Utilities
| Class  | Description                   |
| ------ | ----------------------------- |
| `p-4`  | `padding: 1rem` on all sides  |
| `pt-2` | `padding-top: 0.5rem`         |
| `pb-6` | `padding-bottom: 1.5rem`      |
| `px-3` | `padding-left/right: 0.75rem` |
| `py-1` | `padding-top/bottom: 0.25rem` |
