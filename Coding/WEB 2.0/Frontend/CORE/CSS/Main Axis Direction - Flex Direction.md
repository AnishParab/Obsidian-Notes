# What?
Defines the **direction of the main axis**.

---
# Values
| Value            | Description                              |
| ---------------- | ---------------------------------------- |
| `row`            | Horizontal (left to right) **(default)** |
| `row-reverse`    | Horizontal (right to left)               |
| `column`         | Vertical (top to bottom)                 |
| `column-reverse` | Vertical (bottom to top)                 |
