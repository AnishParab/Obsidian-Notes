# Font Properties

| Property          | Use Case                                                                |
| ----------------- | ----------------------------------------------------------------------- |
| `font-family`     | Specifies the typeface for text, with fallback fonts for compatibility. |
| `font-size`       | Sets the size of the text to control hierarchy and readability.         |
| `font-weight`     | Adjusts the thickness of text (light, normal, bold, etc.).              |
| `font-style`      | Defines the style of text (normal, italic, oblique).                    |
| `line-height`     | Controls vertical spacing between lines for readability.                |
| `letter-spacing`  | Adjusts space between individual characters.                            |
| `word-spacing`    | Adjusts space between words.                                            |
| `text-transform`  | Changes case of text (uppercase, lowercase, capitalize).                |
| `text-decoration` | Adds or removes decorations like underline, overline, or strikethrough. |
| `text-align`      | Aligns text horizontally (left, right, center, justify).                |
| `text-shadow`     | Adds shadow effects to text for depth or emphasis.                      |
| `white-space`     | Controls how whitespace inside an element is handled.                   |
| `font-variant`    | Controls usage of small caps or normal text.                            |
| `direction`       | Sets text direction (LTR or RTL).                                       |
