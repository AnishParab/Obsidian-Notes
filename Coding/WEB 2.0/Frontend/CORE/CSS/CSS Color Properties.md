# Color Properties

| Property                | Use Case                                                                 |
| ----------------------- | ------------------------------------------------------------------------ |
| `color`                 | Sets the **text color** of an element.                                   |
| `background-color`      | Defines the **background fill color** of an element.                     |
| `border-color`          | Specifies the color of an element’s border.                              |
| `outline-color`         | Sets the color of the outline drawn outside the element’s border.        |
| `text-decoration-color` | Changes the color of underlines, overlines, or line-through decorations. |
| `column-rule-color`     | Sets the color of the line drawn between multi-column layouts.           |
| `caret-color`           | Specifies the color of the text cursor (caret).                          |
| `fill` _(SVG)_          | Defines the fill color inside SVG shapes.                                |
| `stroke` _(SVG)_        | Specifies the color of the stroke (outline) of SVG shapes.               |
