# Difference Between the Three

| Feature                                 | **Block**                                  | **Inline**                          | **Inline-Block**                       |
| --------------------------------------- | ------------------------------------------ | ----------------------------------- | -------------------------------------- |
| **Starts on new line?**                 | ✅ Yes                                      | ❌ No                                | ❌ No                                   |
| **Takes full width?**                   | ✅ Yes (by default)                         | ❌ No (fits content)                 | ❌ No (fits content)                    |
| **Can set width/height?**               | ✅ Yes                                      | ❌ No                                | ✅ Yes                                  |
| **Respects vertical margin/padding?**   | ✅ Yes                                      | ❌ Minimal effect                    | ✅ Yes                                  |
| **Respects horizontal margin/padding?** | ✅ Yes                                      | ✅ Yes                               | ✅ Yes                                  |
| **Common tags**                         | `<div>`, `<p>`, `<h1>`–`<h6>`, `<section>` | `<span>`, `<a>`, `<strong>`, `<em>` | `<img>`, `<button>`, `<input>`         |
| **Use case**                            | Page layout & containers                   | Styling text within content         | Elements in a line with precise sizing |
