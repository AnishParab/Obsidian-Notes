# What is `display: inline-block`?
- A **CSS display value** where elements behave **like inline elements** (sit next to each other) **but** can have **width**, **height**, and full **margin/padding** control.
- Best of both worlds: inline flow + block box model.

---
# Key Characteristics

| Behavior                         | Description                                                           |
| -------------------------------- | --------------------------------------------------------------------- |
| **Flows inline**                 | Stays on the same line as text or other inline/inline-block elements. |
| **Width/height allowed**         | Can explicitly set dimensions.                                        |
| **Respects all margins/padding** | Both horizontal and vertical spacing work.                            |
| **Box model applied**            | Treated like a block for sizing and spacing.                          |
