# Syntax

> Short-hand: **WSC**

``` css
border: <border-width> <border-style> <border-color>;

```

Example :

| Property        | Example                              |
| --------------- | ------------------------------------ |
| `border-top`    | `border-top: 3px dotted green;`      |
| `border-right`  | `border-right: 5px solid blue;`      |
| `border-bottom` | `border-bottom: medium dashed gray;` |
| `border-left`   | `border-left: 4px double purple;`    |

---
# Border Properties

| Property                    | Use Case                                                                                                                         |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `border`                    | Shorthand to set **width**, **style**, and **color** in one line.                                                                |
| `border-width`              | Sets the thickness of the border (e.g., `thin`, `medium`, `thick`, or in units like `px`).                                       |
| `border-style`              | Defines the borderâ€™s appearance (`solid`, `dashed`, `dotted`, `double`, `groove`, `ridge`, `inset`, `outset`, `none`, `hidden`). |
| `border-color`              | Specifies the color of the border.                                                                                               |
| `border-radius`             | Rounds the corners of the border.                                                                                                |
| `border-top`                | Sets the border for the top side only (can include width, style, color).                                                         |
| `border-right`              | Sets the border for the right side only.                                                                                         |
| `border-bottom`             | Sets the border for the bottom side only.                                                                                        |
| `border-left`               | Sets the border for the left side only.                                                                                          |
| `border-image`              | Uses an image as the border instead of a solid color.                                                                            |
| `border-collapse` _(table)_ | Determines if table borders are collapsed into a single border or separated.                                                     |
| `border-spacing` _(table)_  | Sets space between table cells when `border-collapse` is `separate`.                                                             |

---
> `border-top-left-radius` ---> This is also allowed.
