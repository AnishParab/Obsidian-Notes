# Syntax
``` css
margin: top right bottom left;

```

Example :

| Example                       | Meaning                                |
| ----------------------------- | -------------------------------------- |
| `margin: 20px;`               | 20px margin on all sides.              |
| `margin: 10px 20px;`          | 10px top/bottom, 20px left/right.      |
| `margin: 5px 10px 15px;`      | 5px top, 10px left/right, 15px bottom. |
| `margin: 5px 10px 15px 20px;` | Top, Right, Bottom, Left (clockwise).  |

---
# Side-Specific Properties

| Property        | Example                |
| --------------- | ---------------------- |
| `margin-top`    | `margin-top: 15px;`    |
| `margin-right`  | `margin-right: 10px;`  |
| `margin-bottom` | `margin-bottom: 20px;` |
| `margin-left`   | `margin-left: 8px;`    |

---
# Special Values

| Value           | Use Case                                                 |
| --------------- | -------------------------------------------------------- |
| `auto`          | Centers block elements horizontally (when width is set). |
| Negative values | Pull elements closer or overlap them.                    |
