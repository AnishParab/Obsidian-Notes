# What is HTTP ? 
- **Hypertext Transfer Protocol**.
- It is an application layer protocol for **transmitting hypermedia documents.** 
- Designed for communication between :-
	- Web Browsers - Web Servers
	- Machine - Machine
	- Programmatic access to APIs
	- and much more.....
- HTTP can also be used to fetch parts of documents to update Web pages on demand.
