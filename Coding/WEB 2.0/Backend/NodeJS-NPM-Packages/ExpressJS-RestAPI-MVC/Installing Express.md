# Make Directory
``` bash
mkdir myapp
$ cd myapp
```

---
# `package.json`
``` bash
npm init

```

---
# Entry Point File
- Make `index.js` or `app.js` as the entry point of your file.

---
# LSP and Dev-Dependencies
``` bash
npm i @types/node -D

npm i @types/express -D
```

---
# Install Express and Save it in Dependencies List
``` bash
npm i express --save

```

---
# To install Express temporarily and not add it to the dependencies list
``` bash
npm i express --no-save

```
