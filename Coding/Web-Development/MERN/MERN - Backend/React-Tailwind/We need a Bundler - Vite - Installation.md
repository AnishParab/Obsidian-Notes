# What is a Bundler ?
- It basically **bundles** my codes (C, Java, TSX, JSX or anything) into plain **HTML, CSS and JS**.
- Examples are **vite, bun** etc.

---
# What is Vite ?
- It bundles React, vanilla-ts, .............. etc.

---
# How to start a vite project ?
**Using npm**
``` bash
npm create vite@latest
```

**Using pnpm**
``` bash
pnpm create vite
```

**Using bun**
``` bash
bun create vite
```
