# Installation using Mason

## `eslint-lsp`

- Linter
- LSP diagnostics
- Code actions

---
## `prettierd`

- Formatter
- Fast daemon-based formatting

---
## `typescript-language-server`

- LSP
- JS/TS IntelliSense

---
## `tailwindcss-language-server`

- LSP
- Tailwind class autocomplete

- HTML/JSX abbreviation expansion

- Node.js debugging
- Chrome DevTools protocol
