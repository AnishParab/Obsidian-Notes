# Deployment Services
## Managed
- **Vercel**
    - Zero-infra management.
    - Optimized for React / Next.js.
    - CI, CDN, TLS handled automatically.
- **Netlify**
    - Managed builds and deployments.
    - CDN-backed static hosting.
    - Serverless functions available.

> They basically run the `build` command.

---
## Self-Managed
- **Server / VM / Amazon EC2**
    - Full control over OS, runtime, networking.
    - You manage builds, processes, scaling, security.
    - Higher operational responsibility, higher flexibility.
