# What is JavaScript?

- Browser understands **HTML, CSS, JavaScript**.
- Browsers contain a **JavaScript engine**.
    - Examples: **V8**, **SpiderMonkey**, **JavaScriptCore (WebKit)**.
- **V8** is open-source and written in C++.
- Ryan Dahl embedded **V8** into a standalone C++ program.
- Result: JavaScript can run **outside the browser**.
- This runtime is called **Node.js**.
- **Node.js = JavaScript runtime environment** (engine + OS APIs).
