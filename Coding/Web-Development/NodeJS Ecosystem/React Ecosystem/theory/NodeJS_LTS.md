# Node.js — LTS

- **LTS (Long-Term Support)** = stable release line with extended maintenance.
- Receives:
    - security fixes
    - critical bug fixes
    - long support window
- Recommended for **production systems**.

---
# **Release Model**

- Even-numbered major versions → enter **LTS** after initial Current phase.
- Odd-numbered major versions → short-lived, experimental/current; not promoted to LTS.
