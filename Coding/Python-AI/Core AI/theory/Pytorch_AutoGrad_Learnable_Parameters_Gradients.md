# AutoGrad
- Automatic Differentiation System.
- It's Pytorch's built-in gradient calculator.

---
> You activate this entire powerful system with one simple switch.

---
# It is
`requires_grad=True`

---
> A tensor is just data.

> To tell pytorch it's a learnable parameter, you must set the switch on.

---
# It tells
> This is a parameter. From now on, **track every single operation** that happens to it.
