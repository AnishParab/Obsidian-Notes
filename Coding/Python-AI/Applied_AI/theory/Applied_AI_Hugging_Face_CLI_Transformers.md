# Docs
[Refer this](https://huggingface.co/docs/huggingface_hub/en/installation)
##### Step 1
- Install hugging face using `pip`. Refer above link.
##### Step 2
- Now do hugging face login which is shown in the **Use this model** section of a model in [Hugging Face Website](https://huggingface.co/google/gemma-3-1b-it). Select **Transformers**.
##### Step 3
- Login will ask for Input tokens.
- In  [Hugging Face Website](https://huggingface.co/google/gemma-3-1b-it),
	- Profile > Access Tokens > Create Tokens
##### Step 4
- Now install this package
``` bash
pip install transformers
```
##### Step 5
- In  [Hugging Face Website](https://huggingface.co/google/gemma-3-1b-it), in **Use this model - Transformer** section, copy paste the codes.
