# Assistant Output History (lowest priority)

**What it is:**  
Previous responses generated by the model in the conversation.

**Purpose:**
- Maintain context
- Enable follow-ups
- Preserve conversational continuity

**Real-life use cases:**
- Continuing a discussion
- Referring back to earlier answers
- Iterative refinement (edits, corrections)

**Example context:**  
“Based on what you said earlier, summarize it in one paragraph.”
