# ChatML vs Alpaca (Quick Comparison)

| Aspect      | ChatML        | Alpaca        |
| ----------- | ------------- | ------------- |
| Format      | Multi-message | Single prompt |
| Roles       | Explicit      | Flattened     |
| Memory      | Supported     | Not supported |
| Fine-tuning | Less ideal    | Ideal         |
| Control     | Strong        | Moderate      |
| Complexity  | Higher        | Simple        |

---
# Mental Model

- **ChatML** = _Conversation protocol_
- **Alpaca** = _Instruction template_
