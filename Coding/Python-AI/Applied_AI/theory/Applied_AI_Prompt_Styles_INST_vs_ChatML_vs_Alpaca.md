# INST vs Alpaca vs ChatML

| Aspect      | INST           | Alpaca          | ChatML     |
| ----------- | -------------- | --------------- | ---------- |
| Origin      | LLaMA-2        | Stanford Alpaca | OpenAI     |
| Tokens      | Special tokens | Plain text      | JSON roles |
| Roles       | No             | No              | Yes        |
| Fine-tuning | Excellent      | Excellent       | Moderate   |
| Chat memory | Manual         | None            | Native     |
| Control     | Moderate       | Moderate        | Strong     |

---
# Mental Model
- **INST** = _Instruction boundary_
- **Alpaca** = _Instruction template_
- **ChatML** = _Conversation protocol_
