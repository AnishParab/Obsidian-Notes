# Arithmetic Operators

| Operator | Description              | Example  | Output |
| -------- | ------------------------ | -------- | ------ |
| `+`      | Addition                 | `2 + 3`  | `5`    |
| `-`      | Subtraction              | `5 - 2`  | `3`    |
| `*`      | Multiplication           | `4 * 3`  | `12`   |
| `/`      | Division (float result)  | `5 / 2`  | `2.5`  |
| `//`     | Floor division (integer) | `5 // 2` | `2`    |
| `%`      | Modulus (remainder)      | `5 % 2`  | `1`    |
| `**`     | Exponentiation (power)   | `2 ** 3` | `8`    |
