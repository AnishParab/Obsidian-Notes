# PyPI (Python Package Index)

- **PyPI** = public registry (index + hosting) for Python packages.
- **PyPI ecosystem** includes:
  - publishers (maintainers)
  - consumers (devs, CI/CD)
  - tools (`pip`, build backends)
  - metadata standards (PEPs)
  - security/trust mechanisms
  - dependency graph

> PyPI is supply chain infrastructure for Python code reuse.
