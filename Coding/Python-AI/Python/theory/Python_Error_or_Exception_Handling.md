# Why ?
- Whenever an error occurs, you don't crash the entire program, but instead you **handle it gracefully**.

---
# Example code of an Error
``` python
orders = ["masala", "ginger"]

# Gives a list index out of range error
print(orders[2])
```

**Output**
``` text
IndexError: list index out of range
```
