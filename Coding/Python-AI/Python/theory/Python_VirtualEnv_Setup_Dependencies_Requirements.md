# Setup (Mac / Linux)
```bash
python3 -m venv .venv
source .venv/bin/activate
```

Verify:
```bash
echo $VIRTUAL_ENV
```

``` bash
pip install --upgrade pip
```


> Now, you can use `python` instead of `python3`.

---
# Dependencies
- `requirements.txt`
```text
requests==2.31.0
flask==3.0.0
```

Install:
```bash
pip install -r requirements.txt
```
