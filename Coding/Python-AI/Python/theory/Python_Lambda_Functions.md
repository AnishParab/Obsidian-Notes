# Code (Lambdas)
> **Lambda = one-line function without a name**
**Syntax**
``` python
lambda arguments: expression
```

``` python
chai_types = ["light", "kadak", "ginger", "kadak"]

strong_chai = list(filter(lambda chai: chai!="kadak", chai_types))

print(strong_Chai)
```

> Note: `map( function , iterable )`

**Output**
``` text
['light', 'ginger']
```
