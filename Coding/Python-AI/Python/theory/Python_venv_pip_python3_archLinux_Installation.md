# 1. Update System
```bash
sudo pacman -Syu
```

---
# 2. Install Python (Latest Stable) - `python3`
```bash
sudo pacman -S python
```

Verify:
```bash
python --version
```

---
# 3. Install pip (usually included)
```bash
pip --version
```

If missing:
```bash
sudo pacman -S python-pip
```

---
# 4. Install venv
```bash
sudo pacman -S python-virtualenv
```
