# Comparison Operators

| Operator | Description              | Example  | Output |
| -------- | ------------------------ | -------- | ------ |
| `==`     | Equal to                 | `3 == 3` | `True` |
| `!=`     | Not equal to             | `3 != 4` | `True` |
| `>`      | Greater than             | `5 > 3`  | `True` |
| `<`      | Less than                | `2 < 4`  | `True` |
| `>=`     | Greater than or equal to | `5 >= 5` | `True` |
| `<=`     | Less than or equal to    | `3 <= 5` | `True` |
