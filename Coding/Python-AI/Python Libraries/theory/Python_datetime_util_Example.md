# Problem Statement
**Challenge: Self-Intro Script Generator**

**Objective**
	- Create a Python script that interacts with the user and generates a personalized self-introduction.
**Requirements**
Your program should:
1. Ask the user for the following details:
   - Name  
   - Age  
   - City  
   - Profession  
   - Favorite hobby  
1. Use the collected information to generate a warm and friendly self-introduction paragraph.
2. Print the final paragraph in a clean and readable format.

**Example**
- User Input
	- **Name:** Priya  
	- **Age:** 22  
	- **City:** Jaipur  
	- **Profession:** Software Developer  
	- **Hobby:** Playing guitar  
- Sample Output
> Hello! My name is Priya. I'm 22 years old and live in Jaipur.  
> I work as a Software Developer and I absolutely enjoy playing guitar in my free time.  
> Nice to meet you!

**Bonus Tasks**
- Append the current date at the end of the paragraph in the following format:  
  `Logged on: YYYY-MM-DD`
- Display the final output wrapped inside a decorative border made of asterisks (`*`).

**Constraints**
- Focus on clarity and readability.
- Use simple input/output operations.
- The output should feel natural and friendly.
