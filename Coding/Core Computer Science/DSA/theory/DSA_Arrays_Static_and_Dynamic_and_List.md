# Static Arrays
- You have to define the size of the array as **prior**.
- `n=9`
- This is used in languages like C, C++.

> Gives an **Exception** if array size is less.

---
# Dynamic Arrays
- No need to define the size of an array prior.
- This is possible in Python.
> If further insertion is needed, it creates a new array of double the size with all the previos elements.

---
# Lists
- Dynamic Array wherin elements can be of different types.
- This is possibl in Python.
