# Row major form

- The elements are stored **row-wise** in the array.
- Acceptable by many languages.

- `[2, 4, 6, 8, 10, 12, 14, 17, 20]`

---
# Column major form

- The elements are stored **column-wise** in the array.

- `[2, 8, 14, 4, 10, 17, 6, 12, 20]`
