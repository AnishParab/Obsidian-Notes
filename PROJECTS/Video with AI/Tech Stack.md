# Tech Stack 
- **Next.js** → Full-stack React framework for building SSR, SSG, and hybrid web apps with API routes.
- **NextAuth.js** → Plug-and-play authentication for Next.js with support for OAuth, JWT, and sessions.
- **MongoDB** → NoSQL database that stores flexible, JSON-like documents for scalable data handling.
- **ImageKit.io** → Real-time image CDN for optimization, transformation, and fast global delivery.
