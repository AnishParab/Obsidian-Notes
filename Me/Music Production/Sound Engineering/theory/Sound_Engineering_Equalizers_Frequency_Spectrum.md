# Frequency Spectrum

- Human hearing range: 20 Hz – 20 kHz
- Each region has a character:

| Region     | Range        | Character                               |
| ---------- | ------------ | --------------------------------------- |
| Sub Bass   | 20–60 Hz     | Felt in the body, not just heard        |
| Bass       | 60–200 Hz    | Thickness, boom                         |
| Low Mids   | 200–600 Hz   | Meat of the mix; muddies fast           |
| Mids       | 600 Hz–3 kHz | Most audible to humans; critical region |
| Upper Mids | 3–8 kHz      | Harshness, brittleness                  |
| Highs      | 8–20 kHz     | Air; absence = muffled sound            |
