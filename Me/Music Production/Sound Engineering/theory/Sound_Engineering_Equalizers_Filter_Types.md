# EQ: Filter Types

- **Bell Filter**
Boosts or cuts a specific frequency range; shape looks like a bell curve; Q controls width

- **High Pass Filter (HPF)**
Lets high frequencies pass; cuts everything below a set point

- **Low Pass Filter (LPF)**
Lets low frequencies pass; cuts everything above a set point

- **Shelf Filter**
Boosts or cuts everything above (high shelf) or below (low shelf) a set frequency; flat effect, not a curve

- **Notch Filter**
Extremely narrow cut at a specific frequency; used to surgically remove problem frequencies (hum, resonance)

- **Band Pass Filter**
Lets only a specific band through; cuts both above and below; HPF + LPF combined

- **Tilt/Slope Filter**
Pivots around a center frequency; boosts one end while cutting the other simultaneously
