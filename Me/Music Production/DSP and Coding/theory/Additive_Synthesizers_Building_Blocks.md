# Building Blocks for Additive Synthesizers

``` text
1. Sine oscillator          ← generate a single partial
2. Partial stack            ← combine N oscillators
3. ADSR envelope            ← shape amplitude over time
4. Filter (lowpass)         ← shape the spectrum
5. Master gain + output     ← render to audio
6. Knobs / live control     ← interactive parameter control
7. Full additive synth      ← wire everything together
```
