##### Build your own VPN (WireGuard)
[[VPN_Why_Build_Your_Own_Service]]
- Install Wireguard using Docker - Visit `wg-easy` github for the setup
- Enter the URI with port to open up wiregurad after the setup
- Log in to wireguard 
- Setup a new client
- For phones you can setup using the QR code using WireGuard app
- Download configuration file for PC
	- Download wireguard vpn client
	- Import tunnel, i.e. the configuration file
	- Click activate
