# Security Threat

- A potential violation of security
- Exists when a circumstance, capability, action, or event could exploit a vulnerability and cause harm

---
# Threat vs Attack

- **Threat** — potential danger (might happen)
- **Attack** — actual attempt (is happening)

---
# What Makes a Threat?

- **Vulnerability** — weakness in the system
- **Capability** — attacker has means to exploit it
- **Intent** — deliberate or accidental

---
# Categories

- **Interception** — unauthorized access to data
- **Interruption** — asset becomes unavailable
- **Modification** — unauthorized alteration
- **Fabrication** — counterfeit data injected

---
# Examples

- Unpatched software → exploitable vulnerability
- Weak password policy → credential theft threat
- No encryption → interception threat
- Single point of failure → availability threat
