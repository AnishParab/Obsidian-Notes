# Brute Force Attack

- Tries every possible key until intelligible plaintext is obtained
- Exhaustive key search — guessing by elimination
- Larger key space = exponentially more time required

---
# Common Tools

- Aircrack-ng, Cain and Abel, Crack, DaveGrohl
- Hashcat, Hydra, John the Ripper
- RainbowCrack, Ophcrack

---
# Countermeasures

- **Larger key size** — increases search space
- **CAPTCHA** — blocks automated login attempts
  - Text-based, ReCAPTCHA, 3D, Mathematical, Image-based
