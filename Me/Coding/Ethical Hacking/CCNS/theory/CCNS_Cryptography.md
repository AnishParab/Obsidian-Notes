# Cryptography

- Process of converting plaintext → ciphertext and back using a key
- Makes intelligible messages unintelligible to unauthorized parties

![[Pasted image 20260612185042.png]]

---
# Key Terms

- **Plaintext** — original readable message
- **Ciphertext** — encrypted, unreadable output
- **Cipher** — the encryption algorithm used for transformation
- **Key** — secret value that controls the cipher's transformation
- **Cryptanalysis** — science of breaking ciphers without the key
- **Cryptology** — Cryptography + Cryptanalysis

---
# Relationship

- Larger key size → exponentially harder to break
- Same plaintext + different key → different ciphertext
