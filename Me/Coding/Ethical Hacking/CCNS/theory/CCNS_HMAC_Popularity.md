# HMAC (Hash-Based MAC)

- Most important practical MAC.

Uses:
```text
Hash Function
+
Secret Key
```

- Instead of designing a new MAC algorithm.
Examples:
```text
HMAC-MD5
HMAC-SHA1
HMAC-SHA256
```

---
# Why HMAC Is Popular

- Fast
- Reuses existing hash algorithms
- Easy implementation
- Industry standard

Used in:
- TLS
- HTTPS
- VPNs
- APIs
