# Symmetric / Private Key Cryptography

- Same key used for both encryption and decryption
- Also called secret key or private key cryptography

---
# Characteristics

- Fast — computationally efficient
- Suitable for large amounts of data
- Both parties must possess the same key
- Key must be shared over a secure channel before communication

---
# Key Distribution Problem

- Main weakness — how to securely share the key?
- If key is intercepted during sharing, all communication is compromised
- Solved by Diffie-Hellman or a Key Distribution Center (KDC)

---
# Examples

- DES, 3DES, AES, Blowfish

---
# Symmetric vs Asymmetric (brief)

- Symmetric — one key, fast, key distribution problem
- Asymmetric — two keys, slow, solves key distribution
