# MAC vs Hash Function

| Feature             | MAC        | Hash Function       |
| ------------------- | ---------- | ------------------- |
| Secret Key          | Required   | Not Required        |
| Authentication      | Yes        | No                  |
| Integrity           | Yes        | Yes                 |
| Confidentiality     | No         | No                  |
| Sender Verification | Yes        | No                  |
| Forgery Resistance  | High       | Low                 |
| Examples            | HMAC, CMAC | MD5, SHA-1, SHA-256 |
