# `node_modules`

- Directory created by `npm` during dependency installation.
- Contains the **actual source code** of installed packages.
- Includes:
    - Direct dependencies
    - Transitive (nested) dependencies

---
# Properties

- Auto-generated → **do not edit manually**.
- Can be deleted and restored using:

```bash
npm install
```

- Structure determined by dependency resolution + lockfile.

---
# Notes

- Not meant for version control (`.gitignore`).
- Exact versions recorded in `package-lock.json`.
