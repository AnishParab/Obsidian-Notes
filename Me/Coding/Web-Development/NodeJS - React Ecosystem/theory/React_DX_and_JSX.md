# Developer Experience (DX)

- **Deterministic**: Same state = same UI.
- **Mental Model**: Focus on the **current state**, not complex transitions.
- **Tooling**: Robust debugging via React DevTools and Error Boundaries.

---
# JSX (Structure + Logic)

Syntactic sugar that colocates UI logic with structure.

- **HTML**: String templates + separate JS files.
- **JSX**: JavaScript functions + nested logic.
- **Example**: `<h1>Hi</h1>` $\rightarrow$ `React.createElement('h1', null, 'Hi')`
