# Edit this :

`package.json`
``` json
{
  "scripts": {
    "start": "node --watch index.js"
  },
}
```

**Now run** :
``` bash
npm start
```

---
# NOTE :

> `--watch` enables native node.js watcher.
>  A watcher monitors filesystem events and restarts the Node process when files change.
