# ECMAScript

- ECMAScript = language specification that JavaScript implements
- Maintained by TC39 committee
- JavaScript = ECMAScript + runtime APIs (browser/Node)

Key versions:

|Version|Year|Notable additions|
|---|---|---|
|ES5|2009|`strict mode`, JSON support|
|ES6 / ES2015|2015|`let/const`, arrow functions, classes, modules, promises|
|ES2017|2017|`async/await`|
|ES2020|2020|optional chaining `?.`, nullish coalescing `??`|
|ES2022|2022|top-level await, class fields|

- New proposals go through TC39 stages (0→4) before becoming spec
- Stage 4 = shipped in next ECMAScript version
