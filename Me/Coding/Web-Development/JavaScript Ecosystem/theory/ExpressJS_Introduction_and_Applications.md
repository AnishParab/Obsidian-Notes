# Express Introduction

- Minimal web framework for Node.js — structured layer over the `http` module
- Adds: routing, middleware pipeline, request/response helpers
- Not a full framework — no ORM, auth, or frontend

| Concept    | Meaning                        |
| ---------- | ------------------------------ |
| Routing    | Map URL → handler              |
| Middleware | Functions executed in sequence |
| Request    | Incoming HTTP data             |
| Response   | Outgoing server reply          |

**Use cases**: REST APIs, backend services, microservices
