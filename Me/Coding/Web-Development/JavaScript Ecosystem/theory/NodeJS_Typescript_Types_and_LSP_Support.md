# Node.js TypeScript Types and LSP Support

```bash
npm install -D @types/node
```

> `-D` is a flag, to install it as a dev-dependency.

---
# Purpose

- Installs **TypeScript type definitions** for Node.js APIs.
- Enables:
    - Static type checking
    - Editor IntelliSense
    - LSP-based autocomplete and diagnostics.

---
# Notes

- `@types/node` provides only **type declarations** — not runtime code.
- Used when writing Node.js projects in TypeScript or with type-aware tooling.
