# `curl` Introduction

> `curl` is a CLI tool backed by `libcurl` — a C library that handles the actual network work. When you run `curl`, you're essentially writing a network request by hand, the same way a browser would — except you control every byte.

- Protocols it supports: HTTP, HTTPS, FTP, SFTP, SMTP, POP3, IMAP, WebSocket, and more.
