# Developer Prompt

**What it is:**  
Application-level rules set by the app builder. Controls behavior, format, scope, and tools.

**Purpose:**
- Shape assistant behavior for a product
- Protect the app from misuse
- Ensure consistent output

**Real-life use cases:**
- Customer support bots with fixed tone and scope
- AI tutors restricted to a syllabus
- Agents forced to output structured data
- Tools-using agents (search, DB, APIs)
- Multi-user apps with shared behavior

**Example context:**  
A fintech app that only allows investment explanations, not advice.
