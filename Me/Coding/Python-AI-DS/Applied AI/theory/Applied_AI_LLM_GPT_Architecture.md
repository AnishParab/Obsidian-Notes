# GPT

- **G**enerative — generates text token by token, no fixed retrieval
- **P**re-trained — trained on large text corpora before task-specific adaptation
- **T**ransformer — self-attention-based neural network
- It is type of *LLM*.
- Objective: predict next token given previous tokens
- From this simple objective it learns grammar, facts, reasoning, code structure
