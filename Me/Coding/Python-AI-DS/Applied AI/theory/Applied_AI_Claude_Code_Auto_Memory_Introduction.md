# Auto-Memory

- Auto-memory is a persistent directory where Claude records learnings, patterns, and insights as it works.

*Example*
- When claude discovers something about your project *("oh, this application uses INR instead of USD")*, it saves that to auto-memory. Next session, it already knows. No more repeating yourself.

---
# NOTE

- Only the first `200 lines` of `MEMORY.md` loads automatically.
