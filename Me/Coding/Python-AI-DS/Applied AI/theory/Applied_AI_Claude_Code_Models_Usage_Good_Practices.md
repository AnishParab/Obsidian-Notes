# Model Usage: Good Practices

A common power-user pattern is to:
- Use **Opus** for the *planning phase* where you are thinking through the architecture, writing specs, making decisions.
- Then _switch_ to **Sonnet** for the *implementation phase* where the thinking is done and you just need reliable code generation.
