# Ollama Models and Claude Code CLI

### Install Ollama

``` bash
curl -fsSL https://ollama.com/install.sh | sh 
```

---
# Run Cloud Models using Ollama in Claude Code

- Now run:
``` bash
ollama launch claude
```

> and select the model you want to launch.
