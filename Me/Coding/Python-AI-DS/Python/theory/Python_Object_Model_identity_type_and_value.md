# Objects
> Everything is an object in python.

---
# `IVT`

Each object has:
- **Identity** → Unique address in memory (`id()`)
- **Type** → The kind of object (`type()`)
- **Value** → The actual data stored

#### Code
``` python
x = 10
print(id(x))      # identity
print(type(x))    # type
print(x)          # value
```

#### Output
``` text
22405776
<class 'int'>
2
```
