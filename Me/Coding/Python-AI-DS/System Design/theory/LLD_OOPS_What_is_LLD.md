# What is LLD ?

> Low Level Design

- LLD = designing the internal structure of a system before writing code
- Focuses on _how_ each component works, not _what_ components exist

**3 Pillars:**
- **Scalability** — design handles growth without restructuring
- **Maintainability** — modify one part without breaking another
- **Reusability** — plug-and-play; zero coupling assumptions between components
