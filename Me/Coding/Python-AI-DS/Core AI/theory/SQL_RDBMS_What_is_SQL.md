# What is **SQL**

- **Structured Query Language**
- SQL is a **declarative, domain-specific language** for interacting with relational databases. You describe _what data you want_; the database engine decides how to retrieve it.
- It is not a general-purpose programming language. There are no loops, no pointers, no memory management. SQL operates entirely on _sets of rows_ (relations) and returns sets of rows.
