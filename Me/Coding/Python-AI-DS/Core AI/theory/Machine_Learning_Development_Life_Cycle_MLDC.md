# Machine Learning Development Lifecycle (MLDC)

1. **Frame the Problem** — define goal, output type, success metric
2. **Gathering Data** — APIs, scraping, databases, surveys
3. **Data Preprocessing** — clean missing values, fix errors, encode categories
4. **EDA (Exploratory Data Analysis)** — visualize distributions, spot patterns, understand the data
5. **Feature Engineering & Selection** — create useful features, drop irrelevant ones
6. **Model Training, Evaluation & Selection** — train candidates, compare via metrics, pick best
7. **Model Deployment** — serve model via API or pipeline into production
8. **Testing** — validate on real-world inputs, check edge cases
9. **Optimize** — tune hyperparameters, retrain on new data, reduce drift
