# Drizzle ORM Introduction

- Lightweight, type-safe ORM for Node.js and TypeScript
- Schema defined in code (TypeScript) — no separate DSL like Prisma
- Supports: PostgreSQL, MySQL, SQLite
- SQL-like syntax — queries look close to raw SQL
- Drizzle Kit — migrations CLI
- No code generation step — schema is directly imported and used
- Trade-off: more verbose than Prisma, but more control and transparency
