# `Drizzle-kit Studio`

**Visual UI on Browser**
- Drizzle Studio is a database browser for Drizzle projects that uses your drizzle config file to connect to your database and lets you browse, add, delete, and update data based on your existing drizzle SQL schema

``` bash
npx drizzle-kit studio
```
