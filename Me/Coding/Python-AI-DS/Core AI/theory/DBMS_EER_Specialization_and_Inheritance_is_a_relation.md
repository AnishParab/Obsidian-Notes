# Specialization

- **Top-down approach**
- One **entity** is divided into **sub-entities** based on distinguishing characteristics

**Property**
- Sub-entities **inherit attributes and relationships**

---
# Inheritance

- Subclasses inherit:
    - **Attributes**
    - **Relationships** - If a parent entity set participates in a relationship then its child entity sets will also participate in that relationship.

---
# Example

![[eer_is_a_relationship.excalidraw|700]]

- _Employee_ is a _Person_ and **inherits** *address* and *name*.
- _Customer_ is a _Person_ and **inherits** *address* and *name*.

> By preventing the attribute *name* in Cutomer and Employee, it prevents **Redundancy**.
