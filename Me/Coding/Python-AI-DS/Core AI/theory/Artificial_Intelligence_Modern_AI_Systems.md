# Era: post-2015 — scale as the dominant axis of progress

**Large-Scale AI Systems**
- Distributed training across hundreds/thousands of GPUs
- Data parallelism, model parallelism, gradient checkpointing
- Enabled training runs infeasible on single machines

**Foundation Models**
- Transformer architecture (Attention Is All You Need, 2017) — self-attention replaces recurrence
- Pretrain on massive corpus → fine-tune or prompt for downstream tasks
- Single model generalizes across many tasks (GPT, BERT, CLIP, etc.)

**Generative Systems**
- GANs — adversarial generator/discriminator training; dominated image synthesis ~2018–2021
- Diffusion models — iterative denoising; surpassed GANs in image quality and stability

**Agent-Based Systems**
- LLMs augmented with planning, memory, and tool use
- ReAct, chain-of-thought, function calling
- → closes gap between language model and acting system
