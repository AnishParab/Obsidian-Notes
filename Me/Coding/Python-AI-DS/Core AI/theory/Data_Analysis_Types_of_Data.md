# Types of Data

- **Quantitative Data** → Numerical
- **Qualitative Data** → Non-numerical (categorical)

---
# Quantitative Data

- Represented in **numerical form**
- Supports **mathematical operations**

**Examples**
- Weight
- Volume
- Cost

---
# Qualitative Data

- Represented as **descriptive categories**
- Does **not support arithmetic operations**

**Examples**
- Name
- Gender
- Hair color
