# Relational Model

- A **Relational Model** = collection of tables (relations)
- Columns = **Attributes** (properties of the entity)
- Rows = **Tuples** (one instance of the entity)

**Example:**

| cust_id | name  | email           |
| ------- | ----- | --------------- |
| 1       | Alice | alice@email.com |
| 2       | Bob   | bob@email.com   |

- Table `Customer` = the **relation**
- `cust_id`, `name`, `email` = **attributes**
- Each row = one **tuple** (one customer instance)
