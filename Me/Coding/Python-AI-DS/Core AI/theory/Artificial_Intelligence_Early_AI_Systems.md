# Era: 1950s–1960s — birth of AI as a field (Dartmouth 1956)

- Dominant paradigm: Symbolic AI / GOFAI (Good Old-Fashioned AI)

**Symbolic AI**
- Intelligence modeled as manipulation of explicit symbols and logical rules
- Assumption: reasoning = formal symbol transformation

**Search & Planning**
- State-space search: represent problem as states, find path to goal
- Heuristics: guide search to reduce combinatorial explosion

**Early Programs**
- Logic Theorist (Newell & Simon, 1956) — proved mathematical theorems using symbolic reasoning
- General Problem Solver (Newell & Simon, 1957) — domain-general problem solving via means-ends analysis
