# Era: 1990s–2000s — shift from hand-crafted rules to learning from data

- Key move: replace explicit knowledge encoding with statistical inference

**Machine Learning Systems**
- Supervised learning — learn mapping from labeled input-output pairs
- Unsupervised learning — find structure in unlabeled data

**Probabilistic Models**
- Bayesian networks — directed graphical models encoding conditional dependencies
- Hidden Markov Models (HMMs) — model sequences with latent state (speech recognition, NLP)

**Ensemble Systems**
- Random Forests — aggregate many decision trees via bagging; reduces variance
- Boosting (AdaBoost, Gradient Boosting) — sequentially correct errors of weak learners; reduces bias
