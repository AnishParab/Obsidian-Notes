# Syntax

- Order of execution is from **Right** to **Left**

``` sql
SELECT <set of column names> FROM <table_name>;
```

---
# SQL Query

**List all Workers**
``` sql
SELECT * FROM Worker;
```

**List only salary and names of Workers**
``` sql
SELECT FIRST_NAME, SALARY FROM Worker;
```
