# Ascending Order using `ORDER BY`

- Arranges data and produces output in ascending order.

---
# SQL Query

**Arrange salaries in ascending order**
``` sql
SELECT * FROM Worker ORDER BY SALARY;
```
OR
``` sql
SELECT * FROM Worker ORDER BY SALARY ASC;
```
