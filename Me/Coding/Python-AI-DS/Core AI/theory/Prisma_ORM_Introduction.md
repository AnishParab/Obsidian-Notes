# Prisma Introduction

- Type-safe ORM for Node.js and TypeScript
- Three components:
    - **Prisma Client** — auto-generated, type-safe query builder
    - **Prisma Migrate** — schema migrations
    - **Prisma Studio** — GUI to inspect/edit DB data
- Schema defined in `schema.prisma` — single source of truth

> Supports: PostgreSQL, MySQL, SQLite, MongoDB, CockroachDB
