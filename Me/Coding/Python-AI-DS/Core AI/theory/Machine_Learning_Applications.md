# Machine_Learning_Applications

- **Retail** (Amazon, Big Bazaar)
    - Buying behavior tracked via phone number → *targeted marketing*
    - `If you're not paying for the product, you are the product`
    - *Product placement* optimized via _association rule mining_
    - Classic example: Walmart study found beer & diapers bought together → placed them adjacent → sales increased

- **Banking & Finance**
    - *Credit/loan risk*: your profile matched against historical defaulters
    - *Fraud detection*: transaction patterns flagged in real time

- **Transport** (Ola, Uber)
    - *Surge pricing* — demand/supply + time + location fed into dynamic pricing model

- **Manufacturing** (Tesla)
    - *Predictive maintenance* — sensor data predicts component failure before it happens; reduces downtime

- **Consumer Internet** (Twitter/X)
    - *Sentiment analysis on tweets* → sell aggregated signals to stock brokers and hedge funds
    - Social data as a financial instrument
