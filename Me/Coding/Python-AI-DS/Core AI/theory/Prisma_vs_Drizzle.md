# Prisma vs Drizzle

| Aspect          | Prisma        | Drizzle     |
| --------------- | ------------- | ----------- |
| Schema          | `.prisma` DSL | TypeScript  |
| Query style     | ORM-like      | SQL-like    |
| Code generation | Yes           | No          |
| Bundle size     | Heavy         | Lightweight |
| Maturity        | Mature        | Newer       |
