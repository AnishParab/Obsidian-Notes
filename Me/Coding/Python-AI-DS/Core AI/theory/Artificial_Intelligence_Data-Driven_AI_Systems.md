# Era: 2010s — deep learning dominance, enabled by data scale + GPU compute

**Deep Learning Systems**
- Neural networks with many layers trained end-to-end on large datasets
- Replaces hand-engineered features with learned representations
- Key enablers: backpropagation + GPUs + ImageNet-scale data

**Computer Vision Systems**
- AlexNet (2012) — broke ImageNet benchmark, triggered deep learning era in CV
- CNNs become standard for image classification, detection, segmentation

**Speech Recognition Systems**
- Deep neural nets replace HMMs as core of ASR pipelines
- Word error rates drop to near-human levels

**Recommendation Systems**
- Collaborative filtering + embeddings at scale
- Deployed widely: YouTube, Netflix, Amazon
