# Era before electronic computing — laying logical and mechanical groundwork

**Formal Logic**
- Aristotle → syllogistic logic (structured deductive inference)
- George Boole → Boolean algebra (logic as mathematical operations)

**Mechanical Computation**
- Blaise Pascal → Pascaline (mechanical arithmetic machine)
- Gottfried Leibniz → Step Reckoner (multiply/divide via repeated addition)

**Symbolic Reasoning Vision**
- Leibniz → "calculus of reasoning" — vision of logic as a formal computational system
- Direct precursor to the idea that thought can be mechanized
