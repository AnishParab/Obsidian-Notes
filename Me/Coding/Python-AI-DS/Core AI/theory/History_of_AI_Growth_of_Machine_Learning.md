# Growth of Machine Learning

> **ML = Theory + Data + Compute**

---
# Before ~2010 (Bottlenecks)

- **Theory**: already existed (stats, optimization)
- **Data**: scarce, hard to collect/label
- **Compute**: limited (CPU-bound)

→ Models known, but not scalable

---
# After ~2010 (Acceleration)

- **Data**: internet → massive, continuous generation
- **Labeling**: crowdsourcing, implicit labels
- **Compute**: GPUs → parallel training

→ Deep learning becomes practical

---
# ~2015–2016 (Inflection)

- Data scale + GPU compute reach usable threshold
- Rapid progress in vision, NLP
