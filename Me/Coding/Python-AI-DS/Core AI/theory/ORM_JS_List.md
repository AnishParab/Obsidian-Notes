# ORM - JS Ecosystem

**Relational (SQL)**

| ORM       | Notes                                          |
| --------- | ---------------------------------------------- |
| Prisma    | Type-safe, auto-generated client, most popular |
| TypeORM   | Decorator-based, supports many SQL DBs         |
| Sequelize | Mature, verbose, widely used                   |
| Drizzle   | Lightweight, SQL-like syntax, gaining traction |
| Knex      | Query builder, not full ORM                    |

---
**Document (MongoDB)**

| ODM      | Notes                |
| -------- | -------------------- |
| Mongoose | De facto standard    |
| Prisma   | Supports MongoDB too |

---
**Redis**

| ODM      | Notes                    |
| -------- | ------------------------ |
| Redis-OM | Object mapping for Redis |

**Other**

| ORM              | DB        | Notes                  |
| ---------------- | --------- | ---------------------- |
