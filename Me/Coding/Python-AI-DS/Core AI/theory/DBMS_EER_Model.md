# Enhanced Entity-Relationship (EER) Model

> **EER Model** → An extension of the **Entity-Relationship (ER) Model** that adds **advanced modeling constructs** to represent more complex data relationships.

---
# Why EER

- Basic ER model is **insufficient for complex real-world systems**
- Requires support for:
    - **Hierarchy**
    - **Inheritance**
    - **Abstraction**
