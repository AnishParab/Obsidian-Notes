# Inplace and Outplace Sorting Algorithms

- **Inplace**: sorts within the original array — $O(1)$ extra space
- **Outplace**: requires additional memory proportional to input — $O(n)$ extra space

---
#### Inplace algorithms

- Bubble, Selection, Insertion, Quick, Heap Sort

---
#### Outplace algorithms

- Merge Sort — needs auxiliary array of size n
- Count, Radix, Bucket Sort — need auxiliary data structures
