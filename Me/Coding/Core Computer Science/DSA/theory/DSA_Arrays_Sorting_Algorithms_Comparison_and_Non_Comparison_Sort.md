# Sorting Introduction

- Arranging elements in a defined order (ascending/descending)
- **Two categories**:
	- *Comparison-based*
	- *Non-comparison-based*

---
### Comparison-based

- Elements are sorted by comparing values against each other
- *Lower bound*: $O(n log n)$ — cannot do better theoretically
- Selection, Bubble, Insertion, Quick, Merge, Heap Sort

---
### Non-comparison-based

- Don't compare elements directly — use properties like digit/frequency
- *Can break* $O(n log n)$ *barrier* → $O(n)$ *possible*
- Count Sort, Radix Sort, Bucket Sort
