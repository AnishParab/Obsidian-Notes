# Problem Statement

- Input: `nums = [2,2,1,1,1,2,2]`
- Majority Element: $frequency > \frac {n} {2}$

**Example**:
- For 2,
	- $frequency = 4$
	- and $\frac {n} {2} = 3$
	- Hence $2$ is a majority element

###### Approach 1
- Sort the array.
- But the **time complexity** using this approach would be $O(n \ log n)$
- And the **space complexity** is $O(1)$

###### Approach 2
- Using a **Hash Data Structure**: *(Dictionary)*
- *(key, value)* pairs
	- $2: 4$
	- $1: 3$
- **Time complexity** would be $O(n)$
- But, the **space complexity** would be $O(n)$
